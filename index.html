Options

Attribute Type Default Description
container Optional Element null Notifications wrapper. Will be created if not specified
default_time Optional number [Int] 3500 Number of miliseconds the notification will be visible if not specified.
vanish_time Optional number [Int] 300 Miliseconds the notification vanish animation will take.
vanish_time Optional number [Int] 300 Miliseconds the notification vanish animation will take.
fps Optional number [Int] 30 Frames per second of progress and vanish animations.
* Optional Object success, error, warning, info You can define your own notification types or override existing ones. For example: 
"friendly": {
    "classes": "alert-friendly",
    "icon": "<svg>...</svg>"
},
"hostile": {
    "classes": "myhostileclass mb-3",
    "icon": "<i class\"icon icon-angry\"><i>"
}

Example 1: Push notification

Don't like it? Check other examples!

Code:
<script src="dist/Notifier.min.js"></script>
<script>
    (function(){
        var notifier = new Notifier();

        document.getElementById("ex1notify1").onclick = function(){
            var notification = notifier.notify("success", "Success notification");
            notification.push();
        };

        document.getElementById("ex1notify2").onclick = function(){
            var notification = notifier.notify("info", "Info notification");
            notification.push();
        };

        document.getElementById("ex1notify3").onclick = function(){
            var notification = notifier.notify("warning", "Warning notification");
            notification.push();
        };

        document.getElementById("ex1notify4").onclick = function(){
            var notification = notifier.notify("error", "Error notification");
            notification.push();
        };
    })();
</script>
